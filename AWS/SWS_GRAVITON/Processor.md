# What is a processor?

## A processor (CPU) is the brain of the computer.

#### Whenever you:

Open a website
Run a Python program
Start a database
Launch a Docker container

the CPU executes all those instructions.

Without the CPU, nothing runs.

## Why are there different processors?

### Just like there are different phone brands:

Samsung
Apple
Google

there are different CPU manufacturers:

Intel
AMD
AWS (Graviton)

All of them do the same job—execute instructions—but they are built differently.


## What does x86 and ARM mean?

### Imagine two people who speak different languages.

Person 1

Speaks English

Person 2

Speaks French

If you give English instructions to the French speaker, they won't understand.

The same thing happens with processors.
